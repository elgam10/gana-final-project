# gana-final-project
Ce dépôt contient un script Bash permettant de calculer l’intérêt simple basé sur trois valeurs fournies par l’utilisateur :
	-	Principal
	-	Taux d’intérêt (%)
	-	Durée (années)

#Formule utilisée.

Intérêt simple = (P × R × T) / 100

# Objectif du projet

Ce projet a été créé dans le cadre du cours “Introduction to Git and GitHub” afin d’apprendre :
	-	La création et la gestion de dépôts GitHub
	-	La rédaction de documentation
	-	Les bonnes pratiques open source
	-	L’utilisation du CLI Git dans un environnement cloud
